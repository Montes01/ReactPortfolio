# React Portfolio

A modern portfolio website built with React, TypeScript, Tailwind CSS, and Vite.

## Tech Stack

- **React** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS** - Styling

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Features

- Responsive design
- Modern UI with Tailwind CSS
- TypeScript for type safety
- Fast development with Vite
